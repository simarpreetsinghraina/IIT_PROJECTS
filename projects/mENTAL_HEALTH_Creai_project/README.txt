#  Mental Health Support Bot

A simple AI-powered mental health support assistant built using **CrewAI**, **Gradio**, and **OpenAI API**. It provides empathetic conversations, mood analysis, and basic self-care suggestions based on user input.

--------------------------------------------------------------------------------------

##  Features

-  Conversational support using LLMs (OpenAI).
-  Multi-agent architecture using CrewAI:
- **Mood Analyzer**: Detects emotional tone from text.
 - **Companion**: Responds with empathy and understanding.
- **Self-Care Recommender**: Suggests activities or tips based on mood.
